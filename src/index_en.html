    About PUZ-PRE v3

    PUZ-PRE v3 is a Web application that is for editing and playing pencil puzzles. Now almost all of supported puzzles are published by nikoli. I develop the script by JavaScript, and also work for implementing HTML5 feature.
    I develop the script mainly with Safari 8.0, test on other browsers.
    to Japanese Menu

    Browsers recommanded

    Firefox (latest version), Google Chrome (latest version), Safari 7.0+, Internet Explorer 10.0+
